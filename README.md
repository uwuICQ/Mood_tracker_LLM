# 🌟 Mood Tracker (Дневник Эмоций)

Mood Tracker — это веб-приложение для осознанного отслеживания своего эмоционального состояния. Пользователь записывает свои мысли, а встроенный анализатор определяет доминирующую эмоцию, её интенсивность и подбирает персональную поддерживающую фразу.

## ✨ Основной функционал

* **Нейросетевой анализ текста:** Определение эмоции (радость, грусть, гнев, любовь, спокойствие и др.) на основе введенного текста с использованием лемматизации (`pymorphy3`).
* **Цветовая визуализация:** Автоматическая генерация уникального градиента в зависимости от интенсивности эмоции.
* **Модальные окна поддержки:** Возврат персональных мотивирующих фраз в ответ на запись.
* **Статистика и Архив:** Сохранение истории записей в базу данных SQLite и вывод агрегированной статистики на интерактивных графиках.
* **Генерация тестовых данных:** Встроенный скрипт для быстрого наполнения базы данных (100+ записей) для демонстрации работы статистики.
* **Экспорт:** Возможность скачать свой дневник в формате `.txt`.

## 🛠 Технологии и Архитектура

**Frontend:**
* HTML5, CSS3, Vanilla JS
* Взаимодействие с сервером через Fetch API (CORS)

**Backend:**
* **Python 3.12**
* **FastAPI** (REST API)
* **Uvicorn** (ASGI-сервер)
* **SQLAlchemy** (ORM для работы с базой данных)
* **SQLite** (Реляционная база данных)
